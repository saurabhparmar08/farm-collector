# farm-collector
This is test.code.farmcollector.controller.a spring boot application exposing REST api to get information about Farm harvesting

# Prerequisites
Java 17

# Getting started locally
## Run Application using command line
Run command:
`./mvnw spring-boot:run`

# Listed APIs
GET /planted -> http://localhost:8080/farm-collector/v1/planted
GET /harvested -> http://localhost:8080/farm-collector/v1/harvested
